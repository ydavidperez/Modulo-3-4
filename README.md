# 🔹 Módulo 3: Enumeración y Escaneo

## 🧠 Descripción
La enumeración es el proceso de obtener información detallada de un sistema después del reconocimiento inicial.

---

## 🔑 Conceptos clave

### 🔍 Escaneo de red
Permite identificar:
- Hosts activos  
- Puertos abiertos  
- Servicios en ejecución  

---

### 🚪 Puertos comunes

| Servicio | Puerto |
|---------|--------|
| HTTP    | 80     |
| HTTPS   | 443    |
| SSH     | 22     |
| FTP     | 21     |

---

### 🧾 Enumeración
Obtención de información como:
- Usuarios  
- Recursos compartidos  
- Versiones de software  

**Técnicas:**
- DNS Enumeration  
- SMB Enumeration  
- SNMP Enumeration  

---

### 🛡️ Detección de vulnerabilidades
Identificación de fallos de seguridad.

**Herramientas:**
- OpenVAS  
- Nessus  

---

### ⚠️ Importancia
- Permite detectar debilidades  
- Base para la explotación  

# 🔹 Módulo 4: Acceso y Explotación

## 🧠 Descripción
Fase donde se intenta acceder al sistema explotando vulnerabilidades.

---

## 🔑 Conceptos clave

### 💥 Explotación
Uso de vulnerabilidades para obtener acceso.

**Ejemplos:**
- Inyección SQL  
- Buffer Overflow  
- Fallos de autenticación  

---

### 🔐 Escalada de privilegios
Obtener permisos de administrador.

---

### 🎯 Payload
Código ejecutado tras la explotación.

**Tipos:**
- Reverse Shell  
- Bind Shell  

---

### 🧰 Herramientas
- Metasploit Framework  
- Hydra  

---

### 🕵️ Persistencia
Mantener acceso al sistema.

---

### 🧹 Borrado de huellas
Eliminar rastros:
- Logs  
- Registros
